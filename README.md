# 업무일지

작업 내용을 작성하면 날짜/시간이 자동으로 기록되는 업무일지 앱입니다.

## 현재 버전 (MVP 1)

- 업무 내용 작성 시 현재 날짜/시간 자동 기록 (`YYYY년 MM월 DD일 HH:mm`)
- 날짜/시간 직접 수정 가능
- 기간(시작일~종료일) 설정 가능
- 목록 조회, 수정, 삭제
- 데이터는 브라우저 `localStorage`에만 저장 (서버 없음)

## 사용 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다. 별도 설치나 빌드 과정이 없습니다.
